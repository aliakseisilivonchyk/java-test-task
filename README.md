# Tech stack:

Language: Java 21 

Frameworks: Spring Boot 4, Spring Data JPA, Spring Web MVC

Messaging: Kafka 

Database: PostgreSQL

Build: Maven, Docker, Docker Compose

# Build and deploy:

- Go to microservices-kafka folder where docker compose config file (compose.yaml) is located.
- To build or rebuild consumer/producers services use `docker compose build`.
- To pull other docker images and create/start docker containers use `docker compose up`.
